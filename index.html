Setup Guide: How to connect live Google Sheets database
  1. Create a new Google Sheet.
  2. Rename the sheet to whatever you want, but the Web App handles creating a dedicated "ValuTrack" tab automatically.
  3. Go to Extensions > Apps Script from the top menu of the Google Sheet.
  4. Clear any existing code in Code.gs and paste the contents of apps_script.js (which is included in this project folder).
  5. Click the Save icon.
  6. Click Deploy > New deployment in the top right.
  7. Click the Gear icon next to "Select type" and choose Web app.
  8. Configure deployment settings:
    • Execute as: Me (your email)
    • Who has access: Anyone
  9. Click Deploy, grant all necessary OAuth permissions, and copy the generated Web app URL.
  10. Paste it into the field above and click "Test Connection", then click "Save Settings".
